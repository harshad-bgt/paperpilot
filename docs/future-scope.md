# Future Scope

While the current iteration of PaperPilot AI provides a powerful foundation for academic discovery, we have identified several key areas for future expansion:

## 1. Multi-Modal Document Parsing
- Integrating OCR and vision models to accurately parse and understand complex mathematical equations, charts, graphs, and tables embedded within PDF research papers.

## 2. Live API Integrations
- Connecting the RAG pipeline directly to live APIs from major publishers (e.g., arXiv, PubMed, IEEE Xplore, Nature) to ensure the knowledge base is updated in real-time.

## 3. Collaborative Workspaces
- Allowing multiple researchers to interact with the same AI agent simultaneously within a shared project space.
- Ability to save AI-generated literature reviews to a shared "Digital Library".

## 4. Automated Grant Proposal Drafting
- Expanding the IBM Granite prompts to help researchers translate their identified research gaps into structured grant proposals and funding applications.

## 5. Personalized Recommendation Engine
- Learning a researcher's specific interests over time to proactively suggest newly published papers that align with their ongoing work.
